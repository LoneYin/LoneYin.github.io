VScode好用插件合集

发表于 2017-11-24 | 分类于 开发环境

前言

    从Sublime Text3转VScode已经有一个多月的时间了。放弃ST3的原因很简单，当时我想装一些关于Node和ES6以及Vue和React的插件， 而ST3这个东西找插件加装插件的过程实在是太麻烦了，不适合我这种频繁折腾的人，Atom我也尝试过，但是但是但是，我对它的界面实在是不感冒。而VScode，个人认为只比ST3丑一点，但是功能却强大了很多倍，而且流畅度完爆Atom，配合各种傻瓜式安装的插件，使用起来真的是十分舒爽。下面就给大家介绍一些十分好用的插件。

常用工具类

Beautify

代码格式化工具，只需一键，还你一段美观整洁的代码，谁用谁知道
下载地址：https://marketplace.visualstudio.com/items?itemName=HookyQR.beautify

Auto Rename Tag

自动匹配闭合标签的修改，意思是你改前面的标签对应的闭合标签也会随着改变，本来还有一个自动闭合标签的插件，但是新版本VScode已经集成该功能。
下载地址：https://marketplace.visualstudio.com/items?itemName=formulahendry.auto-rename-tag

background

Duang~Duang~Duang~
这个插件可以给你的编辑器添加背景，快去找一张png格式的小姐姐放在你的编辑器里面吧！具体用法见插件详情
下载地址：https://marketplace.visualstudio.com/items?itemName=shalldie.background

HTML Class Suggestions

根据工作空间内的所有CSS文件来给HTML文件添加class类名的提示和补全。
下载地址：https://marketplace.visualstudio.com/items?itemName=AndersEAndersen.html-class-suggestions

HTML CSS Support

根据工作空间内的所有CSS、SCSS文件来给HTML文件添加class类名和id属性的提示和补全。
下载地址：https://marketplace.visualstudio.com/items?itemName=ecmel.vscode-html-css

Sass

拓展VScode对sass、scss文件的支持，提供语法高亮
下载地址：https://marketplace.visualstudio.com/items?itemName=robinbentley.sass-indented

Easy Sass

对工作空间内的sass、scss文件进行自动编译，导出css文件及map文件
下载地址：https://marketplace.visualstudio.com/items?itemName=spook.easysass

SCSS IntelliSense

根据工作空间内的所有文件来给scss文件提供class和id的自动补全
下载地址：https://marketplace.visualstudio.com/items?itemName=mrmlnc.vscode-scss

CSScomb

CSS样式排序，具体使用方法见插件详情
下载地址：https://marketplace.visualstudio.com/items?itemName=mrmlnc.vscode-csscomb

View In Browser

看名字就知道它是做什么的了吧
下载地址：https://marketplace.visualstudio.com/items?itemName=qinjia.view-in-browser

Live Server

开启一个本地server并预览
下载地址：https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer

JS工具类

JavaScript(ES6) code snippets

为VScode提供ES6语法支持和代码提示
下载地址：https://marketplace.visualstudio.com/items?itemName=xabikos.JavaScriptSnippets

jQuery Code Snippets

超过一百三十多种的jQuery代码提示，jq党的福音
下载地址：https://marketplace.visualstudio.com/items?itemName=donjayamanne.jquerysnippets

jshint

JS的语法风格检查工具
下载地址：https://marketplace.visualstudio.com/items?itemName=dbaeumer.jshint

npm Intellisense

npm模块路径补全
下载地址：https://marketplace.visualstudio.com/items?itemName=christian-kohler.npm-intellisense

Path Intellisense

路径补全
下载地址：https://marketplace.visualstudio.com/items?itemName=christian-kohler.path-intellisense

React/Redux/react-router Snippets

非常好用的React全家桶代码提示
下载地址：https://marketplace.visualstudio.com/items?itemName=discountry.react-redux-react-router-snippets

Vetur

Vue语法支持及各种功能
下载地址：https://marketplace.visualstudio.com/items?itemName=octref.vetur

Vue 2 Snippets

Vue.js 2+ 的代码提示
下载地址：https://marketplace.visualstudio.com/items?itemName=hollowtree.vue-snippets

ESLint

可定制的JS语法风格检查工具，很强大
下载地址：https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint

Babel JavaScript

Babel 提供各版本js(包括jsx)的语法高亮
下载地址：https://marketplace.visualstudio.com/items?itemName=mgmcdermott.vscode-language-babel

主题和图标

One Dark Pro

饱受好评的Atom主题，在VScode上一样好看
下载地址：https://marketplace.visualstudio.com/items?itemName=zhuangtongfa.Material-theme

seti-icons

超级好看的一套图标，强烈推荐
下载地址：https://marketplace.visualstudio.com/items?itemName=qinjia.seti-icons

持续更新中
